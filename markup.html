HTML

O sistema WWW usa texto marcado para representar um documento de hipertexto para transmissão pela rede. A linguagem de marcação de hipertexto é um formato SGML.

Os analisadores WWW devem ignorar tags que não entendem e ignorar atributos que não entendem de tags que entendem.
